# 📊 Comprehensive Analysis of EdTech StartUps in India — Power BI Dashboard

## 🗂️ Project Overview
This project presents a comprehensive Power BI dashboard analyzing 3,000+ online courses 
scraped from leading EdTech platforms like Coursera, Udemy, and edX. The goal is to uncover 
actionable insights on viewer engagement, category demand, language distribution, and content 
strategy to help EdTech startups make data-driven decisions.

---

## 📁 Dataset
- **File:** `Online_Courses.csv`
- **Total Records:** 3,000+ (after data cleaning)
- **Source:** EdTech platforms (Coursera, Udemy, edX, etc.)
- **Key Fields:** Category, Sub-Category, Course Type, Language, Subtitle Languages,
  Rating, Number of Viewers, Duration, Instructors, Site

---

---

## 📌 Key Insights

- 🥇 **Data Science** is the most in-demand category with **6,326 avg viewers/course**
- ⭐ **Average rating of 4.70/5** reflects high content quality across platforms
- 👥 **8 Million total registrations** with 3,210 avg registrations per course
- 🎓 **Specializations** attract higher average viewers than standalone courses
- 🌐 **97.33% of courses are in English** — major opportunity for regional language content in India
- 📝 **More subtitle languages = more views** — accessibility drives engagement
- ⏱️ **Short-to-medium courses (0–200 hrs)** attract the highest viewer count

## 💡 Business Recommendations

- Launch **Data Science, IT & Computer Science** content first for maximum reach
- Offer courses as **Specialization bundles** for higher engagement
- Add **multilingual subtitles** to all courses to boost accessibility
- Keep course duration **concise (under 200 hours)** for better viewer retention
- Tap into **regional/Hindi language** content — a massive untapped market in India
- Partner with **top-rated instructors** early to ensure content quality

## 🚀 How to Use

1. Clone or download this repository
2. Open `Mini_power_Bi_project.pbix` in **Power BI Desktop**
3. Ensure `Online_Courses.csv` is in the same folder as the `.pbix` file
4. Refresh the data if prompted
5. Explore all 10 analysis pages using the bottom navigation tabs
